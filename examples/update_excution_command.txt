下一步按这个顺序做，最稳：

## 1) 先完成一次性环境设置（Windows + Ollama）

在 **PowerShell** 执行：

```powershell
setx OLLAMA_FLASH_ATTENTION 0
setx OLLAMA_CONTEXT_LENGTH 32768
setx OLLAMA_NUM_PARALLEL 1
setx OLLAMA_MAX_LOADED_MODELS 1
```

然后：
- 彻底退出 Ollama（托盘图标 `Quit`）
- 重新启动 Ollama
- 开一个**新终端**（`setx` 对新进程生效）

---

## 2) 检查项目 `.env`（不是 `.env.template`）

确保 `.env` 至少有这些值：

```env
LLM_BASE_URL=http://127.0.0.1:11434/v1
LLM_API_KEY=ollama
LLM_MODEL=qwen3.5:35b
EMBEDDING_MODEL=bge-m3:567m
EMBEDDING_DIM=1024
LLM_CONTEXT_LENGTH=32768
LLM_JSON_MODE=1
```

---

## 3) 做两个快速连通性检查（建议）

```powershell
python -c "from openai import OpenAI; import os; from dotenv import load_dotenv; load_dotenv(); c=OpenAI(base_url=os.getenv('LLM_BASE_URL'), api_key=os.getenv('LLM_API_KEY','ollama')); r=c.chat.completions.create(model=os.getenv('LLM_MODEL'), messages=[{'role':'user','content':'Reply exactly: OK'}]); print(r.choices[0].message.content)"
```

```powershell
python -c "from openai import OpenAI; import os; from dotenv import load_dotenv; load_dotenv(); c=OpenAI(base_url=os.getenv('LLM_BASE_URL'), api_key=os.getenv('LLM_API_KEY','ollama')); v=c.embeddings.create(model=os.getenv('EMBEDDING_MODEL'), input=['hello']).data[0].embedding; print('dim=',len(v))"
```

---

## 4) 跑冒烟测试（分两步）

先只跑 LightRAG：

```powershell
python examples/e2e_smoke_test.py --limit 1 --skip-raganything
```

再跑完整（含 RAGAnything）：

```powershell
python examples/e2e_smoke_test.py --limit 1
```

---

## 5) 如果 RAGAnything 仍失败，先单独验证 MinerU

```powershell
mineru -p "data/raw/papers/unparsed/<一篇小pdf>.pdf" -o "data/processed/mineru_preflight" -m auto
```

如果这里失败，再按日志处理 `MINERU_MODEL_SOURCE` / `HF_TOKEN`。
