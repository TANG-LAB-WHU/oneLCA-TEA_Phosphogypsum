# Configuration Directory

This directory contains YAML configuration files that define the parameters, settings, and factors used by the PG-LCA-TEA framework.

## Key Files

- **`settings.yaml`**: Global application settings, including default units, paths, and calculation switches.
- **`country_profiles.yaml`**: Regional data for major PG-producing countries (e.g., labor costs, electricity prices, grid mix).
- **`impact_factors.yaml`**: Characterization factors for Life Cycle Impact Assessment (LCIA) based on standards like ILCD/ReCiPe.
- **`shadow_prices.yaml`**: External cost values for societal LCA (SLCC) calculations, defining the monetary value of environmental impacts.

## Modification Guide

Users can modify these files to customize the assessment for specific project needs or update the framework with new research data (e.g., updated carbon prices).
