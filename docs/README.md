# Documentation Directory

This directory contains technical documentation and manuals for the PG-LCA-TEA framework.

## Contents

- **API Reference**: Documentation for the `pgloop` core modules.
- **Methodology**: Detailed explanation of the LCA, TEA, and Risk assessment logic.
- **User Guide**: Instructions for setting up and running analyses.
- **Developer Guide**: Information on how to extend the framework with new pathways or chemicals.

## Format

Documentation is primarily written in Markdown. Rich documentation (e.g., HTML) can be generated using tools like Sphinx or MkDocs if configured.
